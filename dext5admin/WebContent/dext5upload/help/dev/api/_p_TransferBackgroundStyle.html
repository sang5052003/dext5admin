DEXT5 Upload :: Config :: TransferBackgroundStyle

config.TransferBackgroundStyle

업로드 전송창을 제외한 배경화면의 스타일을 지정합니다.

remarks

웹표준모드에서만 동작합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 전송창을 제외한 배경화면의 스타일을 지정합니다.
        DEXT5UPLOAD.config.TransferBackgroundStyle = 'background-color:black;';

        new Dext5Upload("upload1");
    </script>
</div>
