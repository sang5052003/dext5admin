DEXT5 Upload :: Config :: ShowEditAlign

config.ShowEditAlign

업로드가 편집모드일 때 버튼 위치를 설정합니다.

remarks

기본값이 "left" 왼쪽 정렬이고, "right"는 오른쪽 정렬입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드가 편집모드일 때 버튼을 오른쪽 정렬합니다.
        DEXT5UPLOAD.config.ShowEditAlign = 'right';

        new Dext5Upload("upload1");
    </script>
</div>
