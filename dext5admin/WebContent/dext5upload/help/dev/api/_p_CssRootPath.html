DEXT5 Upload :: Config :: CssRootPath

config.CssRootPath

사용자가 만든 css 폴더 경로를 업로드 영역에 설정합니다.

remarks

dext5upload > css 폴더 안에 존재 하는 파일명은 유지 시켜주셔야 합니다. (파일명 변경 불가)
http://를 포함하는 전체 url을 입력해야 합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 사용자가 만든 css_admin 폴더 경로를 설정합니다.
        DEXT5UPLOAD.config.CssRootPath = 'http://localhost/css_admin/';

        new Dext5Upload("upload1");
    </script>
</div>
