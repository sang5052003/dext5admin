DEXT5 Upload :: Config :: DefaultDownloadPath

config.DefaultDownloadPath

다운로드 시 저장경로를 설정합니다.

remarks

1. 경로 앞에 “*”를 붙이면 사용자가 설정한 경로를 무시하고 해당경로로만 저장됩니다.
예) “*D:\Temp”

2. “*”가 없이 경로를 입력하면 사용자가 기본저장 폴더를 설정하지 않은 경우에 적용 됩니다.
예) “D:\Temp”

3. “-“ 값 입력 시 설정한 defaultDownloadPath가 삭제 됩니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 다운로드 시 저장경로를 설정합니다. 
        DEXT5UPLOAD.config.DefaultDownloadPath  = 'D:\\temp';

        new Dext5Upload("upload1");
    </script>
</div>
