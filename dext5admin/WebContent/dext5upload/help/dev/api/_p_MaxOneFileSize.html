DEXT5 Upload :: Config :: MaxOneFileSize

config.MaxOneFileSize

업로드할 때 파일당 용량제한을 설정합니다.

remarks

기본값은 "0" 용량 제한없음이고, "사용자 설정값" 으로 설정합니다.
파일 단위는 B(byte), KB(kilobyte), MB(megabyte), GB(gigabyte) 으로 설정합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 할 파일당 최대 크기를 10MB로 제한합니다.
        DEXT5UPLOAD.config.MaxOneFileSize = '10MB';

        new Dext5Upload("upload1");
    </script>
</div>
