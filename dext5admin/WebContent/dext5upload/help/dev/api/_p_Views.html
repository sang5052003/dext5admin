DEXT5 Upload :: Config :: Views

config.Views

업로드 파일 목록 보기 형태를 설정합니다.

remarks

기본값은 "list" 입니다.
"list"로 설정시 "리스트형태로 보기", "thumbs" 로 설정시 "썸네일형태로 보기" 입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 파일 목록 보기 형태를 썸네일보기로 설정합니다.
        DEXT5UPLOAD.config.Views = 'thumbs';

        new Dext5Upload("upload1");
    </script>
</div>
