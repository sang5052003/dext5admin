DEXT5 Upload :: Config :: ButtonBarPosition

config.ButtonBarPosition

업로드가 편집/보기모드일 때 버튼바 영역 위치를 설정합니다.

remarks

기본값은 "bottom"(하단) 이고, "top"(상단) 설정시 전송영역 맨 위에 버튼바가 설정됩니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 버튼바를 상단에 위치합니다.
        DEXT5UPLOAD.config.ButtonBarPosition = 'top';

        new Dext5Upload("upload1");
    </script>
</div>
