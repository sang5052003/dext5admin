DEXT5 Upload :: Config :: FileNameRule

config.FileNameRule

서버에서 저장할 파일의 이름을 지정하는 규칙입니다.

remarks

기본값은 "REALFILENAME" 입니다.
값 지정은 "GUID" 또는 "REALFILENAME" 을 입력합니다.
"REALFILENAME" 의 경우 파일의 실제 파일명으로 설정됩니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 파일의 이름을 서버에 실제 파일명으로 저장합니다.
        DEXT5UPLOAD.config.FileNameRule = 'REALFILENAME';

        new Dext5Upload("upload1");
    </script>
</div>
