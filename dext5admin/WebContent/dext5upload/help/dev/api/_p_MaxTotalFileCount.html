DEXT5 Upload :: Config :: MaxTotalFileCount

config.MaxTotalFileCount

업로드 할 최대 파일 개수 제한값을 설정합니다.

remarks

기본값은 "0" 제한없음이고, "0 보다 큰 수" 설정시 해당 개수 초과로 첨부 할 수 없습니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 할 최대 파일 개수를 5개로 제한합니다.
        DEXT5UPLOAD.config.MaxTotalFileCount = '5';

        new Dext5Upload("upload1");
    </script>
</div>
