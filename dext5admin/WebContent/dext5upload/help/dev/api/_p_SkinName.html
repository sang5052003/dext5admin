DEXT5 Upload :: Config :: SkinName

config.SkinName

업로드의 스킨 색상을 설정합니다.

remarks

기본값은 "blue" 색상으로 설정 됩니다.

스킨 색상은 "blue, brown, darkgray, gold, gray, green, orange, pink, purple, red, silver, yellow" 12가지 입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드의 스킨을 gray 로 설정합니다.
        DEXT5UPLOAD.config.SkinName = 'gray';

        new Dext5Upload("upload1");
    </script>
</div>
