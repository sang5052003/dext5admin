DEXT5 Upload :: Config :: SilentUpload

config.SilentUpload

전송창 노출 여부를 설정합니다.

remarks

기본값은 "0" 노출이고, "1"로 설정시 미노출입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 전송창을 미노출합니다.
        DEXT5UPLOAD.config.SilentUpload = '1';

        new Dext5Upload("upload1");
    </script>
</div>
