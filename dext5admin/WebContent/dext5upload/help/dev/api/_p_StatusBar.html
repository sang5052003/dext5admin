DEXT5 Upload :: Config :: StatusBar

config.StatusBar

업로드의 상태바를 표시하거나 숨기는 기능을 설정합니다.

remarks

기본값은 "1" 이고, "0" 으로 설정시 상태바를 표시하지 않습니다.

StatusBarShowLimit  "파일 제한" 문구를 표시하거나 숨기는 기능을 설정합니다.
기본값은 "1" 이고, "0" 으로 설정시 "파일 제한" 문구를 표시하지 않습니다.

StatusBarShowStatus  "현재 파일 상태" 문구를 표시하거나 숨기는 기능을 설정합니다.
기본값은 "1" 이고, "0" 으로 설정시 현재 파일 상태 문구를 표시하지 않습니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드의 상태바를 미사용으로 설정합니다.
        DEXT5UPLOAD.config.StatusBar = "0";

        // 업로드 파일 제한 문구를 미표시 합니다.
        DEXT5UPLOAD.config.StatusBarShowLimit = "0";

        // 업로드 현재 파일 상태 문구를 미표시 합니다.
        DEXT5UPLOAD.config.StatusBarShowStatus = "0";

        new Dext5Upload("upload1");
    </script>
</div>
