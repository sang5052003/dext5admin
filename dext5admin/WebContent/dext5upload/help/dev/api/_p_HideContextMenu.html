DEXT5 Upload :: Config :: HideContextMenu

config.HideContextMenu

업로드 context menu(우 클릭 메뉴) 사용 여부를 설정합니다.

remarks

기본값은 "0" 사용이고, "1"로 설정시 context menu(우 클릭 메뉴) 미사용을 설정합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // context menu(우 클릭 메뉴) 미사용으로 설정합니다.
        DEXT5UPLOAD.config.HideContextMenu = '1';
    
        new Dext5Upload("upload1");
    </script>
</div>
