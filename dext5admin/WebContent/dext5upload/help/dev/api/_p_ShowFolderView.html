DEXT5 Upload :: Config :: ShowFolderView

config.ShowFolderView

업로드 폴더 구조로 업로드/다운로드 사용을 설정합니다.

remarks

기본값은 "1" 이고 폴더로 진행, "0"으로 설정시 폴더 올리기 시 파일로 보여집니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 폴더 구조로 업로드/다운로드 미 사용을 설정합니다.
        DEXT5UPLOAD.config.ShowFolderView  = '0';

        new Dext5Upload("upload1");
    </script>
</div>
