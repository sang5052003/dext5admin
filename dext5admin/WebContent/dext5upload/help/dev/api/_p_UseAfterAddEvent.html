DEXT5 Upload :: Config :: UseAfterAddEvent

config.UseAfterAddEvent

업로드 리스트 컨트롤에서 파일 추가 후 이벤트 함수 사용을 설정합니다.

remarks

기본값은 "1" 사용이고, "0"으로 설정시 이벤트 미사용합니다.
"1"으로 설정 후 파일 추가 후 DEXT5UPLOAD_AfterAddItem(uploadID, strFileName, nFileSize, nAddItemIndex, objFile) 함수가 호출 됩니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<script type="text/javascript">
    
<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 파일 추가 후 이벤트 함수 미사용으로 설정합니다.
        DEXT5UPLOAD.config.UseAfterAddEvent = '0';
    
        new Dext5Upload("upload1");
    </script>
</div>
