DEXT5 Upload :: Config :: Width

config.Width

업로드의 넓이를 설정합니다.

remarks

기본값은 "800px" 입니다.
넓이는 "px", "%" 모두 가능합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드의 너비값을 500px 으로 설정합니다.
        DEXT5UPLOAD.config.Width = '500px';

        new Dext5Upload("upload1");
    </script>
</div>
