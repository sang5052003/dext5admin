DEXT5 Upload :: Config :: Timeout

config.Timeout

서버로부터 응답이 지연 되는 경우 이어받기 시도 시간을 설정합니다.

remarks

기본값은 "0" 이고, 이어받기 시도 시간을 "사용자 설정값"으로 설정합니다.
이어받기 시도 시간은 millisecond(시간의 단위)로 설정됩니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 이어받기 시도 시간을 사용으로 설정합니다.
        DEXT5UPLOAD.config.Timeout  = '0';

        new Dext5Upload("upload1");
    </script>
</div>
