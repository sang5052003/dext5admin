DEXT5 Upload :: Config :: SizeColumnWidth

config.SizeColumnWidth

업로드 영역 항목 중 파일크기 항목 노출 여부를 설정합니다.

remarks

설정값 "0" 파일크기 항목 노출안함이고, "1"로 설정시 파일크기 항목을 노출합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 파일크기 항목 노출안함으로 설정합니다.
        DEXT5UPLOAD.config.SizeColumnWidth = '0';
    
        new Dext5Upload("upload1");
    </script>
</div>
