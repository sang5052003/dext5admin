DEXT5 Upload :: Config :: Mode

config.Mode

업로드를 편집모드, 보기모드로 설정합니다.

remarks

기본값은 "edit" 편집모드로 설정됩니다.
"edit or 빈값"이면 "편집모드"이고, "view"로 설정시 보기모드 입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드를 보기모드로 설정합니다.
        DEXT5UPLOAD.config.Mode = 'view';

        new Dext5Upload("upload1");
    </script>
</div>
