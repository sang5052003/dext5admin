DEXT5 Upload :: Config :: FileNameRuleEx

config.FileNameRuleEx

업로드 되는 파일명이 중복될 때 파일명 뒤에 기호를 추가하여 생성됩니다.

remarks

기본값은 "_" 입니다.
기호는 "_" 또는 "#" 으로 설정하면 해당 문자가 파일명 뒤에 붙어 저장됩니다.
값을 "i" 로 설정시 파일명 뒤에 "숫자"가 자동 증가합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 파일명이 중복될 때 숫자로 설정합니다.
        DEXT5UPLOAD.config.FileNameRuleEx = "i";

        new Dext5Upload("upload1");
    </script>
</div>
