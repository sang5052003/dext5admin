DEXT5 Upload :: Config :: SilentDownload

config.SilentDownload

다운로드 전송창 노출 여부를 설정합니다.

remarks

기본값은 "0" 노출이고, "1"로 설정시 미노출, "2"로 설정시 일시정지,취소버튼 미노출(플러그인)입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 다운로드 전송창을 미노출합니다.
        DEXT5UPLOAD.config.SilentDownload = '1';

        new Dext5Upload("upload1");
    </script>
</div>
