DEXT5 Upload :: Config :: Height

config.Height

업로드의 높이를 설정합니다.

remarks

기본값은 "200px" 입니다.
최소 높이값은 "120px" 입니다.
높이는 "px" 만 가능합니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드의 높이값을 400px 으로 설정합니다.
        DEXT5UPLOAD.config.Height = '400px';

        new Dext5Upload("upload1");
    </script>
</div>
