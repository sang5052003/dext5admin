DEXT5 Upload :: Config :: CompleteEventResetUse

config.CompleteEventResetUse

업로드 완료 이벤트에서 ResetUpload API 사용여부를 설정합니다.

remarks

설정값 "0"은 사용안함이고, "1"로 설정시 사용입니다.

sample code

<script type="text/javascript" src="Dext5Upload/js/dext5upload.js"></script>	

<!-- ..... 생략 ..... -->
 
<div style="width:800px;height:200px">  
    <script type="text/javascript">
        // 업로드 완료 이벤트에 ResetUpload API을 사용으로 설정합니다.
        DEXT5UPLOAD.config.CompleteEventResetUse  = '1';

        new Dext5Upload("upload1");
    </script>
</div>
