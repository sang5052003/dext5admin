DEXT5 Editor :: Config :: UserFieldValue

config.UserFieldValue

파일 업로드에 사용되는 User 태그의 value값을 설정합니다.

remarks

필요 시 DEXT5.config.UserFieldValue = "newValue" 으로 설정합니다.
UserFieldID와 같이 사용할 수 있습니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 에디터의 User 태그의 value를 dext5로 변경합니다.
        DEXT5.config.UserFieldValue = "dext5";

        new Dext5editor("editor1");

    </script>       
</div>
