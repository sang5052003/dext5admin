DEXT5 Editor :: Config :: zXssRemoveTags

config.zXssRemoveTags

에디터에 입력되면 삭제할 태그를 설정합니다.

remarks

script, iframe, object 등의 형태로 태그를 설정합니다.
사용자가 해당되는 태그를 입력한 경우 에디터에서 삭제합니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터에 script가 입력되면 삭제합니다.
        DEXT5.config.zXssRemoveTags = "script";

        new Dext5editor("editor1");
    
    </script>       
</div>
