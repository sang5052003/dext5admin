DEXT5 Editor :: Config :: zTopMenu

config.zTopMenu

에디터 메뉴바를 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 메뉴바를 숨김으로 설정하기를 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "0" 이고 메뉴바 보임, "1" 로 설정 시 숨김으로 설정합니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터의 메뉴바를 숨김으로 설정합니다.
        DEXT5.config.zTopMenu = "1";

        new Dext5editor("editor1");

    </script>       
</div>
