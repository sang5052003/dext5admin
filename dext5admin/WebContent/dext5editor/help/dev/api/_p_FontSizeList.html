DEXT5 Editor :: Config :: FontSizeList

config.FontSizeList

에디터 기본 크기 메뉴에 노출할 크기 리스트를 설정합니다.

remarks

기본 크기는 "12pt" 입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 에디터 크기 메뉴에 노출할 크기를 10, 11 으로 설정합니다.
        DEXT5.config.FontSizeList = "10,11";

        new Dext5editor("editor1");

    </script>       
</div>
