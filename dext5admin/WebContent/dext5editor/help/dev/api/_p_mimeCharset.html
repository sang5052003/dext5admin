DEXT5 Editor :: Config :: mimeCharset

config.mimeCharset

MIME에 사용할 문자 인코딩(character encoding) 형태를 설정합니다.

remarks

기본값은 "utf-8" 입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 인코딩 형태를 euc-kr 로 설정합니다.
        DEXT5.config.mimeCharset = "euc-kr";

        new Dext5editor("editor1");

    </script>       
</div>
