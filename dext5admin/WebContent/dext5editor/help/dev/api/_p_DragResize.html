DEXT5 Editor :: Config :: DragResize
DEXT5 Editor :: Config :: DragResizeApplyBrowser

config.DragResize

에디터 또는 브라우저의 기본 리사이즈 사용여부를 설정합니다.

remarks

기본값은 "0" 이고 브라우저 기본 리사이즈를 사용, "1"로 설정 시 에디터에서 제공하는 리사이즈를 사용합니다.

"1" 일 경우 DragResizeApplyBrowser 설정이 가능합니다.
DragResizeApplyBrowser에 입력한 브라우저만(ie,edge,firefox,opera,chrome,safari) 리사이즈를 사용합니다.
빈값일 경우 모든 브라우저가 설정됩니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // 에디터에서 제공하는 리사이즈를 사용 설정합니다.
        DEXT5.config.DragResize = "1";

        // 에디터에서 제공하는 리사이즈를 ie,firefox,opera 브라우저만 사용합니다.
        DEXT5.config.DragResizeApplyBrowser = "ie,firefox,opera";

        new Dext5editor("editor1");

    </script>       
</div>
