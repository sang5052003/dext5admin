DEXT5 Editor :: Config :: SkinName

config.SkinName

에디터의 스킨 색상을 설정합니다.

remarks

기본값은 "blue" 색상으로 설정됩니다.
필요 시 DEXT5.config.SkinName = "색상명" 으로 설정합니다.

스킨 색상은
blue, brown, darkgray, gold, gray, green, orange, pink, purple, red, silver, yellow, random
12가지 입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 에디터의 스킨을 gray 로 설정합니다.
        DEXT5.config.SkinName = "gray";

        new Dext5editor("editor1");
    
    </script>           
</div>
