DEXT5 Editor :: Config :: ViewModeAutoHeight

config.ViewModeAutoHeight

View 모드 시 에디터 안의 내용만큼 자동으로 height가 늘어납니다.

remarks

기본값은 "0" 이고 자동 height 미사용, "1"로 설정 시 사용으로 설정합니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        // 에디터 자동 height를 사용으로 설정합니다.
        DEXT5.config.ViewModeAutoHeight = "1";

        new Dext5editor("editor1");

    </script>       
</div>
