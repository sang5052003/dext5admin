DEXT5 Editor :: Config :: LineHeightList

config.LineHeightList

에디터 줄간격 메뉴에 노출할 줄간격 리스트를 설정합니다.

remarks

기본 줄간격은 "120%" 입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        // 에디터 줄간격 메뉴에 노출할 줄간격을 130%, 140% 으로 설정합니다.
        DEXT5.config.LineHeightList = "130%,140%";

        new Dext5editor("editor1");

    </script>       
</div>
