DEXT5 Editor :: Config :: mimeUse

config.mimeUse

MIME 사용 여부를 설정합니다.

remarks

기본값은 "0" 이고 MIME 미사용, "1" 로 설정 시 사용입니다.

sample code

<script type="text/javascript" src="dext5/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        // MIME 을 사용으로 설정합니다.
        DEXT5.config.mimeUse = "1";

        new Dext5editor("editor1");

    </script>       
</div>
